# Spring_Project

### **프로젝트 명: 병원예약**

## **기획의도**

1. 병원 진료를 예약 할 수 있는 프로그램.

2. 병원을 갔었는데 병명을 클릭하더니 그와 관련된 증상들이 나오는 걸 보고 영감을 얻었다.

ex) 감기를 눌렀을 경우 - 열, 가래, 코막힘, 기침 등등

## **벤치마킹**

1. 서울대학교 병원, 천안 단국대학교 병원, 인하대학교 병원

## **주요기능**

1. 회원예약

2. 예약내역 확인

- 회원

아이디, 비밀번호를 확인

3. . 중복(시간) 예약 불가능

4. 예약취소

## **그외 기능**

1. 의료진 소개

2. 다음API를 통해 병원 위치를 보여줌.

3. 댓글에 별점사용
  - 전체평균값이 나옴.

## **프로젝트를 마치며**

1. 예약기능은 내가 생각했던것 보다 더욱더 어려웠고 생각할게 너무 많았다.

2. 이번 프로젝트를 진행하면서 초기 계획이 얼마나 중요한지를 너무나도 뼈저리게 느꼈다.

3. 중간중간 계속 막혀서 무력감이 들었지만 그래도 주변 사람들을 잘 만나서 힘을 얻을 수 있었다.

결론: 초기 계획을 잘 잡고 차근차근 해나가며 시간이 많다고 느껴질때 하나라도 더 하자.
